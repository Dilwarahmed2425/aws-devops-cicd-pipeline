# AWS DevOps CI/CD Pipeline

## Project Overview

This project demonstrates an end-to-end CI/CD pipeline using:

- Git & GitHub
- Jenkins
- Maven
- Docker
- Docker Hub
- Kubernetes
- AWS EC2

## Architecture

Developer
→ GitHub
→ Jenkins
→ Maven Build
→ Docker Build
→ Docker Hub
→ Kubernetes Deployment
